# 言語化演習 回答

## Q1. 非制御がなぜ再レンダリングに効くか

**回答：**
RHFは入力値の状態をステートで管理するのではなく、DOMに任せています。ステートで管理する場合、入力値が変更したタイミングで再レンダリングが実施されるので、基本的に全てのコンポーネント、周囲にあるコンポーネントがレンダリングされますが、React Hook Formでは入力値を変更したところで再レンダリングが走らないため、use stateを使った管理よりもパフォーマンスという部分で優れているかと思います。

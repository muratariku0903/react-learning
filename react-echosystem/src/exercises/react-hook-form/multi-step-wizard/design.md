ShouldUnregisterはfalseを設定しておく。RFHV7の場合はデフォルトでfalseとなっている。理由としては、多段フェーズになっているので、ページ遷移したタイミングで前のフェーズがアンマウントされてしまうので、アンマウントされたときでも入力した値っていうのは維持しておく必要があるためです。最終フェーズの登録にて、今までずつ重なってきたフェーズの値をすべて登録リクエストに添える必要があります。

また最終フェーズでは、GetValuesとUseWatchの2択で、GetValuesを使うことにした。UseWatchだと毎回レンダリングが走ってしまうため、一方でGetValuesは何かしらのRFHのフックが走らない限り再レンダリングされないため。

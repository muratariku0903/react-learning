# 言語化演習 回答

## Q1. 両者の違いを整理する

<!--
stopPropagation() と preventDefault() の違いについて、以下の観点で比較してください。
- 何を止めるのか
- 親要素への影響
- 使用する典型的な場面
-->

**回答：**

| 観点 | stopPropagation() | preventDefault() |
|------|-------------------|------------------|
| 何を止めるのか | 親要素へのイベント伝播 | ブラウザのデフォルト動作 |
| 親要素への影響 | 親要素にイベントが伝わらない | 親要素に限った話ではない |
| 典型的な場面 | 親要素と小要素に別々のイベントが設定する（選択したらイベントが発生するカードでさらに内部に何かしらの別のアクションが発生するボタンが設置されてる） | フォームを送信した際に、ページをリロードさせない |

---

-->

**回答：**

a要素のデフォルト挙動によりページ遷移する


### stopPropagation() を呼ばなかった場合

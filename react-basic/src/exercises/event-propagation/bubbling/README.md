## 目的

## 実装演習

現在の実装には問題があります。以下の要件を満たすように修正してください。

### 現状の問題

`App.tsx` には、カードコンポーネントとその中の削除ボタンがあります。
現在、削除ボタンをクリックすると、以下の両方が発生してしまいます：

1. カード全体の選択処理
2. 削除確認のアラート

期待する動作は、削除ボタンをクリックしたときは「削除確認」だけが表示されることです。

### 要件

1. 削除ボタンをクリックしたときは、カードの選択が発生しないようにする
2. カードの他の部分をクリックしたときは、カードが選択される
3. 設計意図を `design.md` に記述すること

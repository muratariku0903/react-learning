### 1. TextInputコンポーネントのProps型定義

```ts
type TextInputProps = {
  label: string;
};
```

### 2. forwardRefの使い方

forwardRefを使ってコンポーネントを定義、第二引数で渡されたrefをラップしてるinput要素に設定する。  
作成したコンポーネントを親から呼び出し、refをpropsとして渡す。

### 3. App.tsx側の設計

- 何個のrefが必要か？
  - 2個（ユーザー名とメールアドレス用）
- ボタンクリック時の処理フロー
  - ボタン押下
  - イベントハンドラ内で、refのフォーカスを更新
  - 子コンポーネントのフォーカスが適応
